## 背景上下文

### pnl mock 字段规模

`mocks/pnlAttributionWorkbench.ts` 导出约 12 个 mock 对象（VolumeRate / TPL / Composition / AnalysisSummary / Carry / Spread / KRD / AdvancedSummary / Campisi），总体约 13.5K 行。每个 mock 里数字字段多。

**务实升级策略**：不手写每个 Numeric literal（`{ raw, unit, display, precision, sign_aware }`），而是用 `formatRawAsNumeric` helper（W1.3 提供，`frontend/src/utils/format.ts`）在 module 顶部构造：

```ts
import { formatRawAsNumeric } from "../utils/format";

// old:
// total_current_pnl: 125_000_000,
// new:
total_current_pnl: formatRawAsNumeric({ raw: 125_000_000, unit: "yuan", sign_aware: true }),

// old:
// current_yield_pct: 0.0425,
// new:
current_yield_pct: formatRawAsNumeric({ raw: 0.0425, unit: "pct", sign_aware: true }),

// old:
// previous_scale: 7_500_000_000,
// new:
previous_scale: formatRawAsNumeric({ raw: 7_500_000_000, unit: "yuan", sign_aware: false }),

// null:
// old: previous_yield_pct: null
// new:
previous_yield_pct: null,   // 保持 null 就行，contracts 里是 Numeric | null
```

### 字段 unit / sign_aware 映射（和 W3.1 一致）

沿用 W3.1 `backend/app/schemas/pnl_attribution.py` 里 `_NUMERIC_FIELDS` 映射表。简化规则：

| 字段名 pattern | unit | sign_aware |
|---|---|---|
| `*_pnl` / `*_effect` / `*_return` / `*_income` / `*_gain` / `*_change` / `carry*` / `rolldown*` / `static_pnl` | `yuan` | `true` |
| `market_value` / `current_scale` / `previous_scale` / `total_market_value` / `market_value_start` / `total_income` | `yuan` | `false` |
| `*_pct` / `*_contribution_pct` / `coupon_rate` / `ytm` / `funding_cost` / `treasury_10y*` / `dr007` / `spread_change` / `yield_change` / `treasury_change` / `ftp_rate` / `static_return` / `*_return_pct` / `primary_driver_pct` / `correlation_coefficient` | `pct` | `true` |
| `duration` / `bucket_duration` / `portfolio_duration` / `tenor_years` | `ratio` | `false` |
| `weight` / `krd` / `curve_slope` | `ratio` | `true`（`weight` 例外，`false`）|
| `portfolio_dv01` | `dv01` | `false` |
| `treasury_10y_total_change_bp` | `bp` | `true` |
| `num_periods` / `num_days` / `bond_count` / `level` / `total_actions` 等 **计数** | **保持 number，不转 Numeric** | |

### Step 2：升级 pnlAttributionWorkbench.ts

在文件顶部加 import：

```ts
import { formatRawAsNumeric } from "../utils/format";
```

然后**逐 mock 对象**把所有数字字段换为 `formatRawAsNumeric({ raw: <原值>, unit: <按映射>, sign_aware: <按映射> })`。

**保留**：`num_periods` / `num_days` / `bond_count` / `level` / `total_actions` 等计数/层级字段（仍然 number）；字符串字段、数组、bool 保持。
