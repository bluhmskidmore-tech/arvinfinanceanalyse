### 关键原则

- **新构造必须带 raw（真正的 float 值）**，不能是 `raw=None`
- **原单位是"元"**（例如 aum 的 `total_market_value_amount` 是 yuan，yi 换算在 display 层做）
- **Numeric.unit 选择：**
  - `aum / ytd / dv01 / AttributionSegment.amount / PnlAttributionPayload.total / ContributionRow.contribution` → `unit="yuan"`
  - `nim / delta 中的百分比变化` → `unit="pct"` （raw 是 **小数**，例如 0.0234 = "+2.34%"）
  - `nim delta 的 point change` → `unit="bp"` （raw 是 bp 数）
  - `RiskSignal.value duration` → `unit="yuan"` + display 自定义（例如 "3.5 年"）—— 或者加一个 unit="yi" ？不对，duration 是"年"。为简洁处理：**RiskSignal.value 的 duration / years-to-maturity 用 `unit="ratio"`**，display 自带 "年" 后缀；**DV01 用 `unit="dv01"`**；**信用集中度 % 用 `unit="pct"`**（raw 是 0-1 的 ratio 还是 0-100 的 pct？看现有代码是 `f"{cred_f:.1f}%"` 且 `cred_f = credit_market_value_ratio_pct` 本身是 percent scale，直接用就是百分比值）。为保持 precision：**RiskSignal.value 一律用 `unit="ratio"` + sign_aware=False**（ratio + display 自带单位后缀，最宽松），这样不强制 raw 是小数 vs 百分比。
- **sign_aware：**
  - `value` 金额大多 `sign_aware=False`（aum 规模、dv01 都是正数）
  - `delta` 变化量 `sign_aware=True`
  - `AttributionSegment.amount` `sign_aware=True`（分段可能负）
  - `PnlAttributionPayload.total` `sign_aware=True`
  - `ContributionRow.contribution` `sign_aware=True`
  - `RiskSignal.value` `sign_aware=False`
- **null 处理：** 使用 `Numeric(raw=None, unit=..., display="—", ...)` 或 `display="无环比"` / `display="未接入"` 表达真缺

### Step 3：改造 format helpers 返回 Numeric

把现有 5 个 format helpers 改为返回 `Numeric` 对象。**保留原函数名**以免破坏其他 imports（如果有的话），内部返回 Numeric：

```python
def _fmt_yi_amount(value: float | None, *, signed: bool = False) -> Numeric:
    """Format a yuan-denominated amount into a Numeric in yi display.

    Retains the original signature to minimize churn at call sites (they just
    receive a Numeric instead of str now; ExecutiveMetric etc. accept both
    thanks to W2.1 coercion, but callers building Numerics directly bypass
    the coerce path).
    """
    if value is None:
        return Numeric(
            raw=None,
            unit="yuan",
            display="—" if signed else "0.00 亿",
            precision=2,
            sign_aware=signed,
        )
    v = float(value)
    yi = v / 1e8
    if signed:
        sign = "+" if yi >= 0 else ""
        display = f"{sign}{yi:,.2f} 亿"
    else:
        display = f"{yi:,.2f} 亿"
    return Numeric(
        raw=v,
        unit="yuan",
        display=display,
        precision=2,
        sign_aware=signed,
    )


def _fmt_signed_segment_yi(yi: float) -> Numeric:
    sign = "+" if yi >= 0 else ""
    return Numeric(
        raw=float(yi) * 1e8,
        unit="yuan",
        display=f"{sign}{yi:.2f} 亿",
        precision=2,
        sign_aware=True,
    )


def _fmt_signed_percent(value: float | None) -> Numeric:
    if value is None:
        return Numeric(raw=None, unit="pct", display="—", precision=2, sign_aware=True)
    sign = "+" if float(value) >= 0 else ""
    return Numeric(
        raw=float(value) / 100.0,  # raw 是 decimal ratio
        unit="pct",
        display=f"{sign}{float(value):.2f}%",
        precision=2,
        sign_aware=True,
    )


def _format_percent_change(current: float | None, previous: float | None) -> Numeric:
    if current is None or previous in (None, 0):
        return Numeric(raw=None, unit="pct", display="无环比", precision=2, sign_aware=True)
    change = ((float(current) - float(previous)) / float(previous)) * 100
    sign = "+" if change >= 0 else ""
    return Numeric(
        raw=change / 100.0,
        unit="pct",
        display=f"{sign}{change:.2f}%",
        precision=2,
        sign_aware=True,
    )


def _format_point_change(current: float | None, previous: float | None) -> Numeric:
    if current is None or previous is None:
        return Numeric(raw=None, unit="bp", display="无环比", precision=2, sign_aware=True)
    change = float(current) - float(previous)
    sign = "+" if change >= 0 else ""
    return Numeric(
        raw=change * 100.0,  # bp = percent point * 100
        unit="bp",
        display=f"{sign}{change:.2f}pp",
        precision=2,
        sign_aware=True,
    )
```

**注意：** `_format_percent_change` 和 `_format_point_change` 原来都返回 "无环比" 或 "XX.XX%/pp"；保持 display 不变，raw 改为真实数值。
