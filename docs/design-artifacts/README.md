# Design Artifacts Navigation

Status label: supporting

This file is navigation only and is non-authorizing. It always defers to
`AGENTS.md` -> `docs/DOCUMENT_AUTHORITY.md` -> `docs/CURRENT_EFFECTIVE_ENTRYPOINT.md`.
If this file conflicts with that chain, follow the authority chain.

## Purpose

This directory stores design references, rendered previews, DOM snapshots, and
implementation notes used as visual or UX evidence. These artifacts support
review and comparison, but they are not product requirements by themselves.

## Use

- Use HTML previews and snapshots as supporting evidence for visual decisions.
- Keep source-of-truth UI behavior in the application code and applicable page
  contracts.
- Preserve artifact provenance when adding new captures or references.
