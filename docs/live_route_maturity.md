# Live Route Maturity Registry

Status label: supporting

This registry is a governance burn-down list for routes marked `readiness: "live"` in `frontend/src/mocks/navigation.ts`.
It does not authorize new metric definitions, source lineage, or formal page truth. Business authority remains with `AGENTS.md`,
`docs/DOCUMENT_AUTHORITY.md`, `docs/page_contracts.md`, `docs/metric_dictionary.md`, and page-specific truth contracts.

## Rules

- `nav_state` mirrors navigation: `live` or `temporary-exception`.
- `maturity_tier` is one of `governed`, `governed-mixed-source`, `candidate`, `temporary-exception`, `placeholder`, or `excluded`.
- `page_contract` must be an existing `PAGE-*` id or an explicit `GAP-*` marker.
- `temporary-exception` rows must carry a real owner, risk, exit condition, review date, and verification path.
- A route may stay live only while its row here remains current.
