### 现有消费点（关键梳理）

`PnlBridgePage.tsx` 中：

1. **KpiCard 块（line 551-584）：** 6 个 KPI 直接读 `summary.total_explained_pnl`、`summary.total_actual_pnl`、`summary.total_residual`；目前用 `cellText()` 和 `toneFromSignedDisplayString()`。需改为读 `.display` + `toneFromNumeric()`。
2. **buildWaterfallOption（line 141-237）：** 用 `chartAxisNumber(summary.total_xxx)` 从字符串提数。需改为 `summary.total_xxx.raw ?? 0`。tooltip 的 `displayStrings` 列表需改为读 `.display`。本次顺便清理 § 8 审计里对 waterfall 的 `Math.*` / `abs` 隐患（审核既有实现应该没有）。
3. **AG-Grid 列（line 286-323）：** 14 个 `{field, ..., type: "numericColumn"}` 列；`field: "beginning_dirty_mv"` 等直接读行字段。后端返回 Numeric 对象后，AG-Grid 默认会把 `{raw, unit, display, ...}` 渲染为 `[object Object]` 并破坏排序。必须引入 valueGetter + valueFormatter。

### Step 3：创建 `pnlBridgeAdapter.ts` + test

在 `frontend/src/features/pnl/adapters/pnlBridgeAdapter.ts` 创建：

```ts
import type {
  ApiEnvelope,
  PnlBridgePayload,
  PnlBridgeRow,
  PnlBridgeSummary,
  ResultMeta,
} from "../../../api/contracts";
import type { DataSectionState } from "../../../components/DataSection.types";

export type PnlBridgeVM = {
  reportDate: string;
  summary: PnlBridgeSummary;
  rows: PnlBridgeRow[];
  warnings: string[];
};

export type PnlBridgeAdapterOutput = {
  vm: PnlBridgeVM | null;
  state: DataSectionState;
  meta: ResultMeta | null;
};

export type AdaptPnlBridgeInput = {
  envelope: ApiEnvelope<PnlBridgePayload> | undefined;
  isLoading: boolean;
  isError: boolean;
};

function readFilterDate(filters: Record<string, unknown> | undefined): string | undefined {
  if (!filters) return undefined;
  const v = filters["report_date"];
  return typeof v === "string" ? v : undefined;
}

function deriveState(
  env: ApiEnvelope<PnlBridgePayload> | undefined,
  isLoading: boolean,
  isError: boolean,
): DataSectionState {
  if (isLoading) return { kind: "loading" };
  if (isError) return { kind: "error" };
  if (!env) return { kind: "empty" };
  const meta = env.result_meta;
  if (meta.vendor_status === "vendor_unavailable") return { kind: "vendor_unavailable" };
  if (typeof meta.source_version === "string" && meta.source_version.includes("explicit_miss")) {
    return { kind: "explicit_miss", requested_date: readFilterDate(meta.filters_applied) };
  }
  if (meta.fallback_mode === "latest_snapshot") {
    return { kind: "fallback", effective_date: readFilterDate(meta.filters_applied) };
  }
  if (meta.vendor_status === "vendor_stale" || meta.quality_flag === "stale") {
    return { kind: "stale", effective_date: readFilterDate(meta.filters_applied) };
  }
  const p = env.result;
  if (!p || (p.rows.length === 0 && p.summary.row_count === 0)) return { kind: "empty" };
  return { kind: "ok" };
}

export function adaptPnlBridge(input: AdaptPnlBridgeInput): PnlBridgeAdapterOutput {
  const state = deriveState(input.envelope, input.isLoading, input.isError);
  const env = input.envelope;
  if (!env || state.kind === "loading" || state.kind === "error") {
    return { vm: null, state, meta: env?.result_meta ?? null };
  }
  const p = env.result;
  const vm: PnlBridgeVM = {
    reportDate: p.report_date,
    summary: p.summary,
    rows: p.rows,
    warnings: p.warnings ?? [],
  };
  return { vm, state, meta: env.result_meta };
}
```

**说明：** VM 没有重命名字段（keep `snake_case` 对齐 row.field → AG-Grid column.field）；adapter 的价值在于 state 派生 + null 安全 + 未来可扩展（例如把 rows 按 quality_flag 分组、提供汇总派生等，目前保持最小）。

对应 test `pnlBridgeAdapter.test.ts`：参考 W5.2-fe template 写 5 条 + 1 条映射验证（loading / error / fallback / explicit_miss / stale / 正常非空）。
