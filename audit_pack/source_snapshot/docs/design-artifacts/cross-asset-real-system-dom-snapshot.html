3）和「设计参考 HTML」差在哪里？

  • 现网页面是 React + inline style + designTokens，不是单文件静态稿；区块更多（研究四卡、传导轴、资产类结论、NCD 代理、联动表、PageOutput）。

  • KPI 带来源 resolveCrossAssetKpis，与桌面包 mock 的 8 固定指标可能不一致。
